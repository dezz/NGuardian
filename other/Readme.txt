1. 動作環境
・Microsoft Windows XP/Vista/7

2. 必要なライブラリ等
・Java Runtime Environment 1.6

3. 使い方
　(1) run.vbsを起動します。
　　　64bit環境の場合は、run(x64).vbsを起動してください。
　　　よくわからない場合はrun.vbsで問題ありません。

　(2) ツール→オプションを選択し、設定ダイアログを開きます。

　(3) 環境に合わせて編集してください。
　　　不明な場合はネットワーク管理者に問い合わせてください。

4. 制限事項
　SMTPAuthやPOP before SMTPなどの認証機構、SSL/TLS接続等には未対応です。
　(いずれ対応したいとは思っています。)

5. その他
　実はMacでも動きます。
